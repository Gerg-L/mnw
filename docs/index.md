## About
Based off the nixpkgs wrapper but:

- in one place
- more error checking
- a sane interface
- `evalModules` "type" checking
- more convenience options
- doesn't take two functions to wrap
- `devMode` hot reloading quicker config iteration
There are no flake inputs.
